## Features

- User & Resource
  - [x] File Management
  - [x] Login with GitHub or Email
- Custom MDX Rendering
  - [x] Interactive Checkboxes
- Workspace
  - [x] Invitation Codes
  - [x] Checkbox Blame
  - [x] Real-time Checkbox State Updates
- Marketplace
  - [x] File Summarization
