## 功能

- 用户 & 资源
  - [x] 文件管理
  - [x] 使用 GitHub 或邮箱登录
- 自定义 MDX 渲染
  - [x] 交互式复选框
- 工作区
  - [x] 邀请码
  - [x] 复选框责任跟踪
  - [x] 实时复选框状态更新
- 市场
  - [x] 文件摘要
